Introduction to HTML

HTML stands for HyperText Markup Language. It is the standard language used to create web pages. HTML describes the structure of a webpage using elements and tags.

Why Learn HTML?

  • It is the basic building block of websites
  • Easy to learn for beginners
  • Used with CSS and JavaScript

Basic Structure of HTML

Every HTML page has a basic structure including <html>, <head>, and <body> tags.

Example:

<html>
  <head>
    <title>My Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
  </body>
</html>
